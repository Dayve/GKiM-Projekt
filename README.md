##Projekt z przedmiotu "Grafika Komputerowa i Multimedia"

### Do zrobienia dwa programy:
 * **Kodowanie do naszego formatu. Trzy argumenty na wejściu:**
   - nazwa pliku
   - 0-1 dla rodzaju kodowania
   - 0-1 dla opcji skala szarości
 * **Dekodowanie do .bmp**
